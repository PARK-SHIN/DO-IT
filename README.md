# 프로젝트 상세내용

## 주제
```
* IT 스터디 웹 서비스
```

## 목적
```
1. DO-IT는 비대면으로 사람들과 모여 같이 스터디를 할 수 있게 개발하였습니다.

2. 공부 시간 기능으로 자신이 공부한 시간을 확인할 수 있으며,
   랭킹을 통해 동기부여가 될 수 있는 기능도 구현하였습니다.
   
3. 단순 스터디만 하는 것이 아니라 회원들에게 질문을 할 수 있는 공간을 만들어
   공부 중 궁금한 부분이나 정보에 대해 서로 공유할 수는 있는 기능을 만들었습니다.
```
