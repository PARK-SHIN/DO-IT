3. 경로
      1) icon, 스터디방 기본 배경
           ->  webapp/resources/images
      2) 페이지 저장 경로
          -> webapp/WEB-INF/views/..
      2) DoIT 접속 경로
          ->http://localhost:8800/Do_IT/

4. 페이지 안 구성요소
      * 헤더,푸터가 있는 모든 페이지는 <body><header><content></content><footer></body>로 구성
      1) 공동 css 추가:
           <!-- 외부 스타일 시트 -->
	<link href='<%= request.getContextPath() %>/resources/css/all.css' rel='stylesheet'>
      2) header 추가 (body 첫째 줄): 
 	<%@ include file='/WEB-INF/views/common/menubar.jsp' %>
      3) 내용 테그는 <content></content>  => 전체 css에 margin적용되어있음
      4) footer 추가 (body 마지막 줄) :
	<footer>
	<%@ include file='/WEB-INF/views/common/footer.jsp' %>
	</footer>	
	
5. 페이지 내 글씨 class 명 지정
      1) 굵은 글씨 class="title"  
	ex. study- 오픈스터디, qna- 새 글 쓰기, shop- 프리미엄 이용권, my- 하루 목표 공부시간

6. nav 클릭 후 css 변경 
  Servlet doGet() 부분에 request.getAttribute("nav1","페이지명"); 추가
  페이지명 : admin, my, study, ranking, shop
